#  FIFA World Cup Analytics Dashboard (Power BI)

## Overview

This project is an interactive Power BI dashboard that analyzes FIFA World Cup tournaments from 1930 to 2018. The dashboard provides insights into tournament history, team performance, match statistics, and squad market values through interactive visualizations.

The project focuses on applying Business Intelligence concepts, data modeling, DAX measures, and dashboard storytelling.

- Matches

- Tournament Standings
- Tournaments
- 2022 National Team Market Values

---

## Key Insights

- The Group Stage contains the largest number of matches and goals.
- Goal scoring has generally increased in modern World Cup tournaments.
- Teams with higher market values often perform well, but market value alone does not guarantee success.
- Historical tournament data provides valuable insights into long-term performance trends.
