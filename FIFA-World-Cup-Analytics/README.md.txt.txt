# FIFA World Cup Analytics Dashboard

## About the Project

This project is an interactive Power BI dashboard that analyzes FIFA World Cup tournaments from 1930 to 2018. The goal was to explore tournament history, team performance, match statistics, and market value through interactive visualizations.

The dashboard was developed as part of my data analytics portfolio to practice data modeling, DAX, and dashboard design using real-world sports data.

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV files
- GitHub

## Dataset

The dashboard combines multiple datasets, including:

- World Cup matches
- Teams
- Tournament information
- Tournament standings
- 2022 national team market values

## Key Insights

Some interesting findings from the analysis:

- Most goals are scored during the Group Stage because it contains the highest number of matches.
- Goal scoring has generally increased in recent World Cup tournaments.
- Teams with higher market values often perform well, but market value alone does not guarantee success.
- Countries with more World Cup appearances tend to achieve stronger overall performance.
